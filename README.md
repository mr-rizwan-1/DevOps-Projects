# DevOps Projects Portfolio

This repository contains a collection of hands-on **DevOps and Cloud Engineering projects** focused on automation, Infrastructure as Code (IaC), and cloud deployment practices.

The projects demonstrate real-world DevOps workflows using tools like Terraform, AWS, Ansible, Docker, and CI/CD pipelines.

## Tools & Technologies

* Terraform
* AWS Cloud
* Linux
* Shell Scripting
* Git & GitHub
* Ansible
* Docker
* CI/CD

## Projects Included

### 1. Terraform AWS Apache Web Server

Provision an Ubuntu EC2 instance on AWS using Terraform and automatically configure it with Apache using a shell provisioning script.

### 2. Terraform VPC + EC2 Infrastructure

Create a custom VPC network including public subnet, route tables, internet gateway, and EC2 instance.

### 3. Terraform Load Balancer Setup

Deploy multiple EC2 instances behind an Application Load Balancer for high availability.

### 4. Terraform Auto Scaling Infrastructure

Implement an Auto Scaling Group with Launch Templates to dynamically scale EC2 instances.

### 5. Terraform Remote State Backend

Configure Terraform remote state storage using AWS S3 and DynamoDB for state locking.

### 6. Terraform + Ansible Server Configuration

Use Terraform to provision infrastructure and Ansible to configure servers automatically.
